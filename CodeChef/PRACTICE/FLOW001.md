# Add Two Numbers

[Problem Link](https://www.codechef.com/problems/FLOW001)

``` c++
#include <bits/stdc++.h>
using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cin.tie(NULL);

	int t, a, b;
	cin >> t;

	for (int i = 0; i < t; i++) {
		cin >> a >> b;
		cout << a + b << endl;
	}

	return 0;
}
```
